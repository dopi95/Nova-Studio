Premier creative production agency in Addis Ababa, Ethiopia. We specialize in professional photography, videography, social media management, and digital marketing solutions.

## 🌟 Features

- **Multilingual Support** - Full English and Amharic (አማርኛ) localization
- **Responsive Design** - Optimized for all devices and screen sizes
- **SEO Optimized** - Multi-language sitemaps and meta tags
- **Modern UI/UX** - Smooth animations and intuitive navigation
- **Contact Integration** - Email, phone, and Telegram direct contact
- **Portfolio Showcase** - Dynamic project gallery
- **Fast Performance** - Built with Vite for lightning-fast load times

## 🚀 Tech Stack

- **Frontend Framework:** React 18.2
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **Routing:** React Router DOM v6
- **Internationalization:** i18next
- **Build Tool:** Vite
- **Email Service:** EmailJS
- **Icons:** React Icons
- **SEO:** React Helmet Async

## 📦 Installation

```bash
# Clone the repository
git clone https://github.com/yourusername/Nova-Studio.git

# Navigate to frontend directory
cd Nova-Studio/frontend

# Install dependencies
npm install

# Start development server
npm run dev
```

## 🛠️ Available Scripts

```bash
npm run dev      # Start development server
npm run build    # Build for production
npm run preview  # Preview production build
```
